# Genomic repeatability and predictability of local (mal)adaptation in a reef-building coral#

This repository contains analysis scripts and data files accompanying our study of genomic signals of local adaptation in Stylophora pistillata coral populations on the Great Barrier Reef.

This work, entitled "Genomic repeatability and predictability of local (mal)adaptation in a reef-building coral" is currently available as a preprint on BioRXiv: https://www.biorxiv.org/content/10.64898/2026.01.17.700041v1

The `Scripts/` folder contains all R, Python and Bash sripts necessary to replicate the analyses. These are organised into ten subfolders and numbered sequencially to reflect the order in which they should be executed. The scripts in this were written for a SLURM-based HPC environment and tested on the University of Queensland's Bunya cluster.

The `Data/` folder contains input, intermediate, and final output files required to replicate the analyses. These files are organised into subfolders mirroring those in Scripts/. Files include sample metadata, environmental data, RDA models, and other key outputs. Raw sequencing data are available on NCBI under BioProject accession PRJNA1404865."
