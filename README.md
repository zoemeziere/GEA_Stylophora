# Genomic repeatability and predictability of local (mal)adaptation in a reef-building coral#

This repository contains analysis scripts and data files accompanying our study of genomic signals of local adaptation in Stylophora pistillata coral populations on the Great Barrier Reef.

This work, entitled "Genomic repeatability and predictability of local (mal)adaptation in a reef-building coral" is currently available as a preprint on BioRXiv: https://www.biorxiv.org/content/10.64898/2026.01.17.700041v1

In the Scripts/ folder, I have included all R, Python and Bash sripts necessary to replicate the analyses. These are organised into ten subfolders and numbered sequencially to reflect the order in which they should be executed. The scripts in this were written for a SLURM-based HPC environment and tested on the University of Queensland's Bunya cluster.

In the Data/ folder, I have included data input files required to run the analyses, as well as intermediate and final output files. These include metadata, environmental data, RDA models etc. Raw sequence data is available on NCBI (BioProject PRJNA1404865).
